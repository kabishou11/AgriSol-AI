# 农光智助·银青共富平台 - 核心开发任务文档

## 项目概述

**项目名称**：农光智助·银青共富平台
**项目定位**：多模态AI驱动的农光互补全年龄段绿色优化助手
**核心理念**：农业 + 能源 + 环境 + 人文四位一体
**技术特点**：本地轻量化部署 + ModelScope API + 专项模型本地推理

## 技术栈选型

### 前端技术栈
- **框架**：Vue 3 + Vite 5
- **UI组件库**：Arco Design Vue（字节跳动出品，现代精美）
- **可视化**：Apache ECharts 5（数据图表）+ Leaflet（地图）
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **HTTP客户端**：Axios
- **工具库**：
  - Day.js（日期处理）
  - Lodash-es（工具函数）
  - VueUse（Vue组合式工具集）

### 后端技术栈
- **运行时**：Node.js 20+
- **框架**：Fastify 4（高性能）
- **数据库**：Better-SQLite3（同步SQLite）
- **AI集成**：
  - OpenAI SDK（ModelScope API兼容）
  - ONNX Runtime（本地模型推理）
  - Sharp（图片处理）
- **工具库**：
  - Zod（数据验证）
  - Winston（日志）
  - Dotenv（环境变量）

### 包管理器
- **pnpm 9+**（快速、节省磁盘空间）

## 关键技术实现要点

### 1. ModelScope API集成
```javascript
// server/src/services/modelscope.js
import OpenAI from 'openai';

export class ModelscopeService {
  constructor() {
    this.client = new OpenAI({
      baseURL: process.env.MODELSCOPE_BASE_URL,
      apiKey: process.env.MODELSCOPE_API_KEY
    });
  }

  async analyzeImage(imageBase64, prompt, useComplexModel = false) {
    const model = useComplexModel
      ? process.env.MODELSCOPE_COMPLEX_MODEL
      : process.env.MODELSCOPE_SIMPLE_MODEL;

    const completion = await this.client.chat.completions.create({
      model,
      messages: [{
        role: 'user',
        content: [
          { type: 'text', text: prompt },
          { type: 'image_url', image_url: { url: imageBase64 } }
        ]
      }],
      max_tokens: parseInt(process.env.MODELSCOPE_MAX_TOKENS),
      temperature: parseFloat(process.env.MODELSCOPE_TEMPERATURE)
    });

    return completion.choices[0].message.content;
  }
}
```

### 2. SQLite数据库封装
```javascript
// server/src/models/database.js
import Database from 'better-sqlite3';
import fs from 'fs';

export class DatabaseService {
  constructor() {
    this.db = new Database(process.env.SQLITE_DB_PATH);
    this.db.pragma('journal_mode = WAL');
    this.initTables();
  }

  initTables() {
    const schema = fs.readFileSync('./src/schemas/database.sql', 'utf-8');
    this.db.exec(schema);
  }

  // 示例：插入作物观测记录
  insertCropObservation(data) {
    const stmt = this.db.prepare(`
      INSERT INTO crop_observations
      (crop_record_id, user_id, image_path, ai_model_used,
       crop_health_score, recommendations)
      VALUES (?, ?, ?, ?, ?, ?)
    `);

    return stmt.run(
      data.cropRecordId,
      data.userId,
      data.imagePath,
      data.aiModel,
      data.healthScore,
      JSON.stringify(data.recommendations)
    );
  }
}
```

## 性能优化策略

### 前端优化
- **路由懒加载**：按需加载页面组件
- **图片懒加载**：使用Intersection Observer
- **虚拟滚动**：长列表使用虚拟滚动
- **组件缓存**：使用`<keep-alive>`缓存页面状态
- **CDN加速**：静态资源使用CDN

### 后端优化
- **数据库索引**：为常用查询字段建立索引
- **JSON缓存**：高频API响应缓存到JSON文件
- **批量处理**：图片分析支持批量上传
- **连接池**：复用HTTP连接
- **压缩响应**：启用gzip压缩

### 弱网优化
