# 快速开发参考手册

## 一、立即开始

### Windows系统
```bash
# 1. 运行初始化脚本
init-project.bat

# 2. 编辑.env文件，配置API Key
# MODELSCOPE_API_KEY=your_key_here

# 3. 启动开发服务
pnpm dev
```

### Linux/Mac系统
```bash
# 1. 赋予执行权限
chmod +x init-project.sh

# 2. 运行初始化脚本
./init-project.sh

# 3. 编辑.env文件
# MODELSCOPE_API_KEY=your_key_here

# 4. 启动开发服务
pnpm dev
```

## 五、核心API速查

### ModelScope API调用示例

```javascript
// server/src/services/modelscope.js
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: process.env.MODELSCOPE_BASE_URL,
  apiKey: process.env.MODELSCOPE_API_KEY
});

// 多模态图片分析
async function analyzeImage(imageBase64, prompt) {
  const completion = await client.chat.completions.create({
    model: "Qwen/Qwen3.5-122B-A10B",
    messages: [{
      role: "user",
      content: [
        { type: "text", text: prompt },
        { type: "image_url", image_url: { url: imageBase64 } }
      ]
    }],
    max_tokens: 2000
  });

  return completion.choices[0].message.content;
}
```

### SQLite数据库操作示例

```javascript
// server/src/models/database.js
import Database from 'better-sqlite3';

const db = new Database('./data/database.sqlite');

// 插入作物观测记录
const insertObservation = db.prepare(`
  INSERT INTO crop_observations
  (crop_record_id, user_id, image_path, crop_health_score)
  VALUES (?, ?, ?, ?)
`);

insertObservation.run(1, 1, '/uploads/crop1.jpg', 85.5);
```

### 前端API调用示例

```javascript
// web/src/api/index.js
import axios from 'axios';

const api = axios.create({
  baseURL: 'http://localhost:3000/api'
});

// 上传图片分析
export async function analyzeCrop(imageFile) {
  const formData = new FormData();
  formData.append('image', imageFile);

  const { data } = await api.post('/crop/analyze', formData);
  return data;
}
```

## 九、常见问题速查

### Q1: pnpm安装失败？
```bash
# 清理缓存
pnpm store prune

# 重新安装
pnpm install
```

### Q2: 端口被占用？
```bash
# 修改.env文件
APP_PORT=3001  # 改为其他端口
```

### Q3: ModelScope API调用失败？
```bash
# 检查.env配置
MODELSCOPE_API_KEY=your_key_here  # 确保填写正确

# 检查网络连接
curl https://api-inference.modelscope.cn/v1/models
```

### Q4: 数据库初始化失败？
```bash
# 删除旧数据库
rm data/database.sqlite

# 重新启动服务（会自动初始化）
pnpm server
```

### Q5: 前端页面空白？
