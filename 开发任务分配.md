# AgriSol-AI 完整开发任务分配

## 设计理念

### UI/UX核心原则
1. **视觉美学**：现代、清新、专业
2. **适老化设计**：大字体、高对比度、简单操作
3. **响应式布局**：完美适配桌面/平板/手机
4. **流畅交互**：动画过渡、即时反馈、加载状态
5. **数据可视化**：直观、美观、易理解

### 配色方案
- 主色：农业绿 `#52c41a`
- 辅色：能源橙 `#fa8c16`、环境蓝 `#1890ff`、人文紫 `#722ed1`
- 中性色：灰度系列
- 背景：浅色模式为主，支持深色模式

### 任务目标

### 具体任务

## 技术要求

### 前端
- 使用Arco Design Vue组件库
- 响应式设计（移动端优先）
- 动画使用Vue Transition + CSS3
- 图表使用ECharts 5
- 状态管理使用Pinia
- API请求使用Axios + 拦截器
- 错误处理和加载状态
- 代码分割和懒加载

### 后端
- RESTful API设计
- 统一响应格式
- 错误处理中间件
- 请求日志
- 数据验证（Zod）
- 图片处理（Sharp）
- 缓存机制
- 性能优化

### 数据库
- SQLite（Better-SQLite3）
- 索引优化
- 事务处理
- 数据备份
